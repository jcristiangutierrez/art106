For this project we were tasked to take we learned from our big 4 circuits. And combine them together on one breadboard in the enclosure of our choosing. We looked at how to add digital inputs and outputs, as well as analog input and outputs. I really like this project seeing what can be created simply by putting some wires together and running some code. The most diffucult process about this project would have to be the soldering it is very tedious and time comsuming, as well as making sure your wires are organized and not getting tangled up. I had plenty of instances where my wires kept overlapping and disconnecting one another, but thankfully for the most part I made my wires really long so they did not get pulled out as much.
